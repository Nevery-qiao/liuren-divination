## 通用排除规则

对于所有项目，通常都要排除：

```gitignore
# OS
.DS_Store        # macOS
Thumbs.db        # Windows
desktop.ini      # Windows

# IDE
.vscode/         # VS Code
.idea/           # JetBrains IDEs
*.swp            # Vim
*.swo            # Vim
*~               # Emacs

# Logs
*.log
logs/

# Environment & Secrets
.env
.env.local
.env.*.local
secrets.json
credentials.json
config.local.*

# Temporary files
*.tmp
*.temp
*.bak
*.cache
```

## 使用建议

1. **项目初始化时就创建** .gitignore
2. **根据项目类型选择合适的模板**
3. **添加项目特定的排除规则**
4. **绝对不要提交敏感信息**（密码、token、密钥等）
5. **可以使用在线生成器**：https://www.toptal.com/developers/gitignore
